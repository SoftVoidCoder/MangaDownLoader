Автоматический скачиватель манги с сайта com-x.life с возможностью сохранения в PDF (с автоматическим разделением по 100 страниц) и апскейлом качества.

## Возможности

### Основные функции
- **Скачивание всей манги** - автоматическое скачивание всех глав манги
- **Режим ожидания** - ожидание кнопки скачивания на странице
- **Автоопределение** - автоматическое определение страниц манги
- **Сохранение сессии** - cookies сохраняются для повторных запусков

### Форматы вывода
- **PDF с разделением** - автоматическое разделение на файлы по 100 страниц
  - Если меньше 100 страниц - один файл
  - Если больше 100 страниц - несколько файлов (part_001.pdf, part_002.pdf и т.д.)
- **CBZ архив** - упаковка всех изображений в zip-архив

### Дополнительные функции
- **Автоматический апскейл** - улучшение качества PDF через Real-ESRGAN
- **Автопоиск драйверов** - автоматическое скачивание geckodriver
- **Графический интерфейс** - удобный интерфейс на PyQt5
- **Настройки** - сохранение конфигурации между запусками
